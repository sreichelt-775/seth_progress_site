Relative humidity tests

20°C data and next steps.

I fitted the Lu (2016) model to Yu's test data initially by eye and then used a solver for refinement.

Using the parameters determined from the Lu (2016) fit, the nonisothermal SWRC now fits the data realistically. As I obtain more data points from my tests, I will follow the same approach using my updated soil dry density and porosity. I also intend to digitize Yu's higher-temperature data and fit the models similarly in preparation for analyzing my own data later. I will combine everything into a single plot once all the data has been collected, and I will fit nonisothermal model (2024) to each dataset.
